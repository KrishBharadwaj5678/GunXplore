# 🔫 GunXplore

Entre no futuro da **realidade aumentada** com o **GunXplore**. Escaneie imagens de armas de fogo e traga instantaneamente modelos 3D detalhados para o seu ambiente, proporcionando uma experiência imersiva.

![GunXploreDemo](https://github.com/KrishBharadwaj5678/GunXplore/raw/main/GunXploreDemo.jpg)

## ✨ Funcionalidades

| **Funcionalidade**          | **Descrição**                                                        |
| --------------------------- | -------------------------------------------------------------------- |
| **🔫 Modelos 3D Realistas** | Explore modelos 3D altamente detalhados de diferentes armas de fogo. |
| **🌍 Realidade Aumentada**  | Interaja com os modelos diretamente no seu ambiente físico.          |

## 🚀 Como Começar

### 1️⃣ Baixar as Imagens dos Marcadores

* Navegue até a pasta **`markers/`** neste repositório.
* Baixe as imagens dos marcadores para o seu dispositivo.
* Esses marcadores são necessários para ativar os modelos no aplicativo.

### 2️⃣ Instalar o Aplicativo

* Baixe e instale o **Aplicativo GunXplore** através do link abaixo:

  **Baixar Aplicativo:**
  [Baixar o aplicativo](https://drive.google.com/file/d/1YQp0ygnplIxYQkf5liDiMSgBa6qnqNSf/view?usp=sharing)

### 3️⃣ Iniciar o Aplicativo

* Abra o aplicativo e conceda as permissões necessárias (acesso à câmera).

### 4️⃣ Escanear os Marcadores

* Aponte a câmera para as imagens dos marcadores baixadas.
* Veja os modelos aparecerem em tempo real.
