# 🔫 GunXplore

Шагните в будущее **дополненной реальности** вместе с **GunXplore**. Сканируйте изображения огнестрельного оружия и мгновенно переносите детализированные 3D-модели в своё пространство для захватывающего интерактивного опыта.

![GunXploreDemo](https://github.com/KrishBharadwaj5678/GunXplore/raw/main/GunXploreDemo.jpg)

## ✨ Возможности

| **Функция**                   | **Описание**                                                                        |
| ----------------------------- | ----------------------------------------------------------------------------------- |
| **🔫 Реалистичные 3D-модели** | Изучайте высокодетализированные 3D-модели различных образцов огнестрельного оружия. |
| **🌍 Дополненная реальность** | Взаимодействуйте с моделями в окружающем вас физическом пространстве.               |

## 🚀 Начало работы

### 1️⃣ Загрузите маркерные изображения

* Перейдите в папку **`markers/`** данного репозитория.
* Скачайте маркерные изображения на своё устройство.
* Эти маркеры необходимы для отображения моделей в приложении.

### 2️⃣ Установите приложение

* Скачайте и установите приложение **GunXplore** по ссылке ниже:

  **Скачать приложение:**
  https://drive.google.com/file/d/1YQp0ygnplIxYQkf5liDiMSgBa6qnqNSf/view?usp=sharing

### 3️⃣ Запустите приложение

* Откройте приложение и предоставьте необходимые разрешения (доступ к камере).

### 4️⃣ Отсканируйте маркеры

* Наведите камеру на загруженные маркерные изображения.
* Наблюдайте, как 3D-модели появляются в реальном времени.
