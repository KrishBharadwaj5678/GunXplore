# 🔫 GunXplore

**GunXplore** で **拡張現実（AR）** の未来を体験しましょう。銃器の画像をスキャンすると、詳細な3Dモデルが即座に現実空間に表示され、没入感のある体験を楽しむことができます。

![GunXploreDemo](https://github.com/KrishBharadwaj5678/GunXplore/raw/main/GunXploreDemo.jpg)

## ✨ 機能

| **機能**           | **説明**                         |
| ---------------- | ------------------------------ |
| **🔫 リアルな3Dモデル** | さまざまな銃器の高精細な3Dモデルを体験できます。      |
| **🌍 拡張現実（AR）**  | 現実の環境内で3Dモデルを配置し、インタラクションできます。 |

---

## 🚀 はじめに

### 1️⃣ マーカー画像をダウンロード

* このリポジトリ内の **`markers/`** フォルダへ移動します。
* マーカー画像をデバイスにダウンロードします。
* これらのマーカーはアプリ内でモデルを表示するために必要です。

### 2️⃣ アプリをインストール

* 以下のリンクから **GunXplore アプリ** をダウンロードしてインストールします。

  **アプリをダウンロード:**
  https://drive.google.com/file/d/1YQp0ygnplIxYQkf5liDiMSgBa6qnqNSf/view?usp=sharing

### 3️⃣ アプリを起動

* アプリを開き、必要な権限（カメラへのアクセス）を許可します。

### 4️⃣ マーカーをスキャン

* ダウンロードしたマーカー画像にカメラを向けます。
* 3Dモデルがリアルタイムで表示される様子をお楽しみください。
